# Sound

## 概要

サウンドに関する機能を提供する基底クラス

サウンド処理に共通するフィールドをもつが、実際の処理は継承先のクラスに依存する。

## フィールド一覧

| 名前     | 概要                           |
| -------- | ------------------------------ |
| tag      | タグ                           |
| filepath | サウンドファイルのファイルパス |
| loop     | サウンドをループするかどうか   |

## メソッド一覧

| 名前  | 概要                   |
| ----- | ---------------------- |
| play  | サウンドを再生する     |
| pause | サウンドを一時停止する |
| stop  | サウンドを停止する     |
