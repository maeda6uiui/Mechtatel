# Component

## 概要

ワールドに配置するオブジェクトの基底クラス

オブジェクトはすべてこのクラスを親にもつ。

## フィールド一覧

| 名前     | 概要               |
| -------- | ------------------ |
| position | 位置(ワールド座標) |
| tag      | タグ               |

## メソッド一覧

| 名前        | 概要             |
| ----------- | ---------------- |
| translate   | 移動             |
| rotX        | X軸回りの回転    |
| rotY        | Y軸回りの回転    |
| rotZ        | Z軸回りの回転    |
| rot         | 任意軸回りの回転 |
| setPosition | 位置の設定       |
| getPosition | 位置の取得       |
| draw        | 描画             |
