# Component2D

## 概要

2Dコンポーネントの基底クラス

2Dに関するコンポーネントはすべてこのクラスを親にもつ。

## フィールド一覧

| 名前     | 概要                          |
| -------- | ----------------------------- |
| position | 位置(2次元座標)               |
| depth    | 深度(-1から1の範囲で指定する) |

## メソッド一覧

| 名前        | 概要                     |
| ----------- | ------------------------ |
| translate   | コンポーネントを移動する |
| rot         | コンポーネントを回転する |
| setPosition | 位置を設定する           |
| getPosition | 位置を取得する           |
| setDepth    | 深度を設定する           |
| getDepth    | 深度を取得する           |
