# Model3D

## 概要

3Dモデルに関する処理を提供するクラス

モデルの読み込みにはAssimpを使用する予定です。

extends Component3D

## フィールド一覧

| 名前     | 概要                         |
| -------- | ---------------------------- |
| filepath | 読み込むモデルのファイルパス |

## メソッド一覧

| 名前         | 概要                                                     |
| ------------ | -------------------------------------------------------- |
| drawMaterial | 指定されたインデックスのマテリアルをもつ面のみを描画する |
